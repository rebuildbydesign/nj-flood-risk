How to Use This Map

  • Toggle Displacement Risk Groups: Use the sidebar buttons to view properties most at risk of flooding and forced relocation.
  • Search for an Address: Use the search bar (top right) to zoom directly to any property in New Jersey.
  • Click on a County: Select any county to view local flood risk and critical infrastructure data in the sidebar.
  • Legend: Use the color key in the sidebar to understand risk group types.
  • Zoom In: Get more detail by zooming in. Some features appear only at higher zoom levels.

For more details on data sources and methodology, visit the NJ Flood Risk = Financial Risk report by Rebuild by Design.
